# voddownloader.user.js

This script requires [Tampermonkey](https://tampermonkey.net/index.php ) plugin to work. Creates a button in top right corner of video, that you can use to download a content of it.

### Support for
- vod.pl
- player.pl
- vod.tvp.pl
- cyfrowa.tvp.pl
- ipla.tv
- cda.pl
